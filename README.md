### Palmtree is an application to create, edit and visualize XML files through a modern GUI

## Requirements

- [node.js >= 7](https://nodejs.org)
- [npm](https://www.npmjs.com/)

## Getting Started
### 1. Install the dependencies 

`$ npm install`

### 2.a. Development purpose

`$ npm run dev`

### 2.b. Build

`$ npm run build`

### 2.c. Test

`$ npm run test`
